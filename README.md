## Project Title
**TechTrend Store – Interactive E-Commerce Website**

## Project Description
TechTrend Store is a simple interactive e-commerce website developed using HTML, CSS, and Vanilla JavaScript.  
The website allows users to browse products, search for products, filter products by category, add items to a shopping cart, update product quantities, remove items from the cart, and complete checkout using a validated form.

The project demonstrates JavaScript DOM manipulation, event handling, arrays and objects, local storage, and error handling using try-catch.

## Features
The system includes the following features:

- Dynamic product display using JavaScript
- Product search functionality
- Product filtering by category
- Add to cart functionality
- Remove product from cart
- Increase and decrease product quantity
- Cart total price calculation
- Cart counter display
- Checkout form with validation
- Data stored using localStorage
- Error handling using try-catch
- Responsive design using CSS
- Product images displayed on the website
## Technologies Used
- HTML
- CSS

## How to Run/View the Project
To run the project:

1. Download or clone the GitHub repository.
2. Open the project folder.
3. Open **index.html** in a web browser.
4. Browse products, add to cart, and test checkout.

No installation is required. The project runs directly in the browser.
